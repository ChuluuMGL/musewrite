# WordPress 博客模板

## 文章结构
```
[引人入胜的标题]
[摘要/导语]

[目录]

[正文]
  - H2 主标题
    - H3 子标题
    - 段落内容
    - 列表/代码块
    
[总结]

[相关阅读]

[标签]
```

## SEO 优化
- 标题包含关键词
- 摘要 150-160 字符
- H2/H3 合理分层
- 内链外链
- 图片 alt 标签
