# 小红书文案模板

## 标题规范
- 不超过 20 字
- 纯文字，不加 emoji
- 包含关键词
- 制造好奇心/痛点/收益

## 正文结构
```
[痛点/场景引入]
↓
[解决方案/核心观点]
↓
[具体步骤/细节]
↓
[总结/行动号召]
```

## 格式要求
- 用 emoji 分隔段落
- 不用 markdown 格式
- 每段 2-3 行
- 总字数 300-800 字

## 标签规范
- 3-5 个标签
- 包含核心关键词
- 包含长尾词

## 示例模板

### 模板 1: 经验分享型
```
📍 [场景描述]

💡 [核心观点]

✅ [步骤 1]
✅ [步骤 2]
✅ [步骤 3]

⚠️ [避坑提醒]

📌 [总结]

#标签 1 #标签 2 #标签 3
```
