# 公众号文章模板

## 标题规范
- 20-30 字
- 包含关键词
- 制造悬念/痛点/收益
- 可加副标题

## 文章结构
```
[引人入胜的开头]
  - 场景描述
  - 痛点引入
  - 数据/案例

[核心内容]
  - 观点 1 + 论据
  - 观点 2 + 论据
  - 观点 3 + 论据

[总结升华]
  - 回顾要点
  - 行动号召
  - 金句收尾
```

## 格式要求
- 段落清晰
- 小标题分层
- 适当加粗重点
- 配图位置标注

## 示例模板

### 模板 1: 观点文
```
标题：[核心观点]

[开头：场景/痛点引入]

01 [小标题 1]
[论述 + 案例]

02 [小标题 2]
[论述 + 案例]

03 [小标题 3]
[论述 + 案例]

[结尾：总结 + 升华]

金句：[一句点睛之笔]
```

### 模板 2: 教程文
```
标题：[教程主题]

[开头：为什么需要这个教程]

一、准备工作
• [准备 1]
• [准备 2]

二、详细步骤
1. [步骤 1]
   [详细说明]
   
2. [步骤 2]
   [详细说明]

三、常见问题
Q: [问题 1]
A: [回答]

[结尾：鼓励行动]
```
