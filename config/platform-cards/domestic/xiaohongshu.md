## 🚨 重要限制（必读）

### 字数限制
- **MCP 接口限制**：≤ 1000 字
- **最佳字数**：500-1000 字
- **超过 1000 字**：会被截断

### 配图规范 ⭐ 关键
- **工具**：Seeddream API（不是 PIL）
- **尺寸**：直接生成 3:4（1800x2400）
- **Prompt**：包含 "vertical portrait composition 3:4 aspect ratio, content centered"
- **❌ 禁止**：生成后裁剪（内容会显示不全）

### 人工审核 ⭐ 关键
- **必须**：展示完整草稿给用户
- **等待**：用户明确回复「确认」或「发布」
- **❌ 禁止**：自动发布（未经确认）

## 发布规范 ⚠️ 重要

### 标题规范
- ❌ **不要 emoji**（标题保持纯文字）
- ❌ 不要 `✅`、`📝` 等符号
- ✅ 直接写内容，如：`达成了 AI 记忆系统的三层改造`
- ✅ 字数：≤20 字

### 正文规范
- ✅ **可以使用 emoji**（正文内容可以）
- ❌ **不要 markdown 格式**（小红书不渲染 markdown）
- ❌ 不要 `**粗体**`、`## 标题`、`- 列表` 等
- ✅ 使用纯文本 + emoji 组织内容
- ✅ 字数：500-1000 字

### 格式转换规则

| Markdown | 小红书格式 |
|----------|-----------|
| `**粗体**` | 直接去掉 `**`，用 emoji 强调 |
| `## 标题` | 换行 + emoji 标题 |
| `- 列表` | 换行 + emoji 开头 |
| `---` | 空行 |
| `[链接](url)` | 直接写 url 或去掉 |

### 转换示例

**Markdown（输入）**：
```markdown
## 三层架构

- hot（心跳必加载）
- warm（30天内）
- cold（归档）

**改造效果**：心跳加载从 ~10KB 降到 ~2KB
```

**小红书（输出）**：
```
📱 三层架构

• hot（心跳必加载）
• warm（30天内）
• cold（归档）

✨ 改造效果：心跳加载从 ~10KB 降到 ~2KB
```

## 关键教训（2026-02-28）

**今天犯的 6 个错误**：
1. ❌ 草稿位置错误 → 必须在 Obsidian 草稿目录
2. ❌ 自动发布（未审核）→ 必须等待用户确认
3. ❌ 配图遗漏 → 发布前必须检查配图
4. ❌ 配图工具错误（PIL）→ 必须用 Seeddream API
5. ❌ 配图生成后裁剪 → 直接生成 3:4 比例
6. ❌ 未经确认自动发布 → "修改草稿" = 只修改，不发布

**关键教训**：
- **用户说"修改草稿" → 只修改，不发布**
- **用户说"确认" → 可以发布**
- **用户说"发布" → 可以发布**
- **其他任何话 → 都不要发布**
