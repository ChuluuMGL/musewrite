# 平台卡 - 抖音

> **优先级**: P0
> **类型**: 短视频
> **更新时间**: 2026-02-28

---

## 发布规范

### 标题规范
- ✅ 可以有 emoji
- ✅ 字数 ≤30 字
- ✅ 吸引眼球

### 正文规范（视频脚本）
- ✅ 60秒脚本约 180-200 字
- ✅ 开头 3 秒抓眼球
- ✅ 口语化，短句为主
- ✅ 结尾有 CTA

---

## 素材规格

| 项目 | 规格 |
|------|------|
| 时长 | 15秒-5分钟 |
| 比例 | 9:16（竖屏） |
| 分辨率 | 1080x1920 |
| 大小 | ≤4GB |
| 帧率 | ≥30fps |

---

## 内容结构（短视频脚本）

```
0-3秒：开头（抓眼球）
  - 问句开头
  - 反差开头
  - 数字开头

3-50秒：主体（干货）
  - 3-5 个要点
  - 每点 10-15 秒
  - 配合画面

50-60秒：结尾（CTA）
  - 总结一句话
  - 引导互动
```

---

## 脚本模板

```
【开头】3秒
你还在用...吗？今天教你...

【主体】45秒
• 第一点...
• 第二点...
• 第三点...

【结尾】12秒
记住了吗？关注我，每天学一招！
```

## 格式转换规则

| Markdown | 抖音脚本 |
|----------|---------|
| `## 标题` | 删除（视频不需要） |
| `- 列表` | 转为口语化表达 |
| `**粗体**` | 删除（强调时语气加强） |

- ❌ 不要长句
- ✅ 口语化
