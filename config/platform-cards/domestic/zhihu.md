# 平台卡 - 知乎

> **优先级**: P2
> **类型**: 图文 + 视频
> **更新时间**: 2026-02-28

---

## 发布规范

### 标题规范
- ✅ 可以有问号（提问式）
- ✅ 字数 ≤50 字
- ✅ 可以更正式

### 正文规范
- ✅ 支持 Markdown
- ✅ 字数无限制（建议 1500-3000 字）
- ✅ 可以更深入、更专业
- ✅ 引用、数据、案例

---

## 素材规格

| 项目 | 规格 |
|------|------|
| 标题 | ≤50 字 |
| 正文 | 无限制 |
| 图片 | 多张，支持长图 |
| 视频 | ≤120 分钟 |

---

## 内容结构

```
开头：背景 + 问题（吸引注意）
  ↓
主体：分析 + 论证（干货）
  - 观点1 + 案例
  - 观点2 + 数据
  - 观点3 + 引用
  ↓
结尾：总结 + 行动（升华）
```

---

## 写作风格

| 特点 | 说明 |
|------|------|
| 专业 | 可以用专业术语 |
| 逻辑 | 结构清晰、论证充分 |
| 引用 | 引用数据、案例、来源 |
| 深度 | 不怕长，怕浅 |

- ✅ 可以引用外部链接
- ❌ 不要标题党
- ❌ 不要营销感
