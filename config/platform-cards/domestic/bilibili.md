# 平台卡 - B站

> **优先级**: P2
> **类型**: 中长视频
> **更新时间**: 2026-02-28

---

## 发布规范

### 标题规范
- ✅ 可以有 emoji
- ✅ 字数 ≤80 字
- ✅ 可以用梗、用网络用语

### 正文规范（视频简介）
- ✅ 支持 Markdown
- ✅ 简介 500 字左右
- ✅ 包含时间轴、相关链接

---

## 素材规格

| 项目 | 规格 |
|------|------|
| 时长 | 3-30 分钟（中视频） |
| 比例 | 16:9（横屏） |
| 分辨率 | 1080p/4K |
| 大小 | ≤8GB |
| 封面 | 16:9，≤2MB |

---

## 内容结构（中长视频脚本）

```
开头（30秒）：自我介绍 + 本期内容预告
  ↓
主体（5-25分钟）：干货内容
  - 分章节
  - 有字幕
  - 配画面
  ↓
结尾（30秒）：总结 + 关注引导
```

---

## 视频脚本模板

```
【开头】30秒
大家好，我是石头哥。
今天我们来聊聊...

【主体】10-25分钟
第一节：背景介绍

第四节：总结

【结尾】30秒
这期视频就到这里。
如果对你有帮助，请三连支持！
我们下期见！
```
