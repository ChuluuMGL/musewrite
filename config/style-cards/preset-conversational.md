# 风格卡 - 对话风格

> **类型**: 预置风格
> **适用场景**: 问答、访谈、观点交流
> **标签**: 对话、互动、真实

---

## 语言风格

### 语气
- 第二人称（你）
- 对话式
- 有互动感

### 特点
- 多用问句
- 直接对话
- 有共鸣

### 结构

```
开头：提出问题
  ↓
主体：逐步解答
  你可能会问...
  其实...
  举个例子...
  ↓
结尾：引发思考
```

---

## 范文

### 示例 1：问句开头

```
你是不是也有这样的困扰：

每天打开电脑，不知道从哪开始？
工具越来越多，效率越来越低？

今天，我想和你聊聊我是怎么解决这个问题的。
```

### 示例 2：对话式展开

```
你可能会问：这真的有用吗？

说实话，我一开始也不信。
但用了 3 个月后，我的工作效率提升了 40%。

怎么做到的？我来详细说说...
```
