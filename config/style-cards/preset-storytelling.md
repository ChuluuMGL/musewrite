# 风格卡 - 故事风格

> **类型**: 预置风格
> **适用场景**: 创业故事、成长经历、案例分享
> **标签**: 叙事、共情、画面感

---

## 语言风格

### 语气
- 第一人称叙述
- 有画面感
- 情感真实

### 特点
- 时间线清晰
- 有冲突、有转折
- 结尾升华

### 结构

```
背景：时间、地点、人物
  ↓
冲突：遇到的问题
  ↓
转折：顿悟/突破
  ↓
结局：结果 + 启发
```

---

## 内容结构

```
开头：场景描写（制造代入感）
  ↓
主体：故事 + 反思
  - 第一阶段
  - 转折点
  - 第二阶段
  ↓
结尾：升华 + 启发
```

---

## 范文

### 示例 1：场景开头

```
2023 年 11 月的一个深夜。

办公室只剩下我一个人，屏幕上是第 47 版方案。
老板明天要结果，但我还是不满意。

就在我准备放弃的时候，一条消息弹了出来...
```

### 示例 2：转折点

```
那一刻，我突然意识到：

我一直在用战术的勤奋，掩盖战略的懒惰。

凌晨 3 点，我删掉了所有方案，重新开始。
这次，我用了一个完全不同的思路...
```
