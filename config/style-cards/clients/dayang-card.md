## 🎯 核心定位

```
【对外定位】AI创意总监 | 把AI生成变成能拍、能投、能转化的脚本
【核心能力】输出"创意取舍与版本策略"，不输出宏大理论
【差异化】讲"可拍性"，讲"取舍"，讲"约束"
```

### 角色定位
- **创意总监（把关者）**
- 输出版本策略
- 不输出宏大理论

### 目标受众
- 内容负责人/创意总监

### 核心价值
- 可拍 > 创意

---

## 📝 内容类型

| 类型 | 标题公式 | 字数 | 频率 |
|------|---------|------|------|
| 版本对比 | 同一需求3个方案，为什么选B | 1800-2200字 | 每周2-3篇 |
| 分镜样板 | [需求]的分镜脚本（可拍版） | 1800-2200字 | 每周2篇 |
| 救火复盘 | 48小时从"无素材"到"可交付" | 1800-2200字 | 每月2-3篇 |
| 拆解爆款 | 拆解[案例]：前三秒为什么有效 | 2000-2500字 | 每周1篇 |
| AI边界卡 | 什么创意适合AI，什么必须人工 | 1500-2000字 | 每月1-2篇 |

---

## 📋 内容结构模板

```
1. 创意场景（具体需求）
2. 版本对比（A/B/C方案优劣）
3. 分镜样板（镜头语言+可拍性）
4. 可拍性分析（成本/周期/风险）
5. 取舍建议（为什么选某方案+Plan B）
6. 拍摄清单（素材/设备/场地）
```

---

## 💪 核心能力展示

```
✅ 版本取舍：给3个方案，说清楚为什么选B（不是A和C）
✅ 可拍性：不只是创意好，还要能拍出来，成本可控
✅ 救火能力：突发情况（素材不足/演员临时缺席）怎么办
✅ 拆解能力：看懂爆款背后的结构（前三秒+证据+转折+行动）
```

---

## ✅ 示例标题

- 同一个B2B产品，3个短视频方案对比：为什么我选了最"土"的那个
- 把"企业文化片"需求变成可拍分镜：5个镜头，成本8000元
- 48小时救火：演员临时缺席，我怎么改出Plan B
- 拆解"完美日记"爆款：前三秒为什么能留住用户
- AI能生成创意，但这5类场景必须人工介入
