## 🎯 核心定位

```
【对外定位】品牌营销避坑指南 | 10年经验帮你少走弯路
【核心能力】输出"判断"，不输出细活
【差异化】用"复核者"身份，帮客户做取舍，不是教客户做事
```

### 角色定位
- **复核者（判断者）**
- 输出判断与取舍
- 不输出执行细节

### 目标受众
- 品牌负责人/市场总监

### 核心价值
- 避坑 > 赋能

## 📝 内容类型

| 类型 | 标题公式 | 字数 | 频率 |
|------|---------|------|------|
| 结论卡 | [场景]最容易踩的3个坑 | 2000-2500字 | 每周2-3篇 |
| 踩坑清单 | 做[X]之前，必须搞清楚的5件事 | 1800-2200字 | 每周1-2篇 |
| 拆解卡 | 拆解[案例]：为什么[X]有效 | 2000-2500字 | 每周1篇 |

---

## 📋 内容结构模板

```
1. 痛点共鸣（场景化开头）
2. 核心判断（明确结论+反驳误区）
3. 可执行策略（3-5步，每步有例子）
4. 边界与风险（适用/不适用场景）
5. 行动指南（立即可做的3件事）
6. 深度案例（真实复盘）
```

## 📢 CTA策略

```
标准版：评论/私信"我的情况"，48小时内给你3条避坑建议
```
