# 风格卡 - 教程风格

> **类型**: 预置风格
> **适用场景**: 工具教程、操作指南、Step-by-step
> **标签**: 实用、清晰、可操作

---

## 语言风格

### 语气
- 直接、简洁
- 步骤清晰
- 避免废话

### 特点
- 分步骤
- 有截图/示例
- 有注意事项

### 结构

```
开头：问题 + 目标
  ↓
主体：分步骤操作
  Step 1
  Step 2
  Step 3
  ↓
结尾：验证 + 进阶
```

---

## 范文

### 示例：分步骤

```
## 三步搞定 XX

### Step 1：准备工作

需要准备：
- 账号 A
- 工具 B
- 配置 C

### Step 2：核心操作

1. 打开 XX
2. 点击 YY
3. 输入 ZZ

⚠️ 注意：这里容易出错，要小心...

### Step 3：验证结果

完成后，检查以下几点：
- [ ] A 正常
- [ ] B 正常
- [ ] C 正常
```
