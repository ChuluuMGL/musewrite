# 风格卡 - 幽默风格

> **类型**: 预置风格
> **适用场景**: 轻松话题、娱乐内容
> **标签**: 幽默、接地气、有梗

---

## 语言风格

### 语气
- 轻松、幽默
- 自嘲式调侃
- 适度玩梗

### 特点
- 多用问句、感叹句
- 适度夸张
- 意想不到的转折

### 禁忌
- ❌ 不要低俗
- ❌ 不要冒犯
- ❌ 不要过度玩梗影响理解

---

## 内容结构

```
开头：自嘲/反差
  ↓
主体：调侃 + 干货
  ↓
结尾：反转/金句
```

---

## 范文

### 示例 1：自嘲开头

```
我承认，我又浪费了 3 小时...

本来想优化一下工作流，结果越优化越乱。
但在崩溃边缘，我突然悟了。

今天分享 3 个我踩过的坑，帮大家少走弯路。
```

### 示例 2：玩梗

```
老板说要"降本增效"。

我心想：这不就是让我一个人干三个人的活吗？
但没想到，AI 真的帮我把三个人的活都干了。

老板：？？？
我：😊
```
