# AI-Writer 反馈日志

> 记录用户对生成内容的反馈，持续优化写作质量

---

## 使用方法

### 添加反馈
```bash
aiwriter feedback add --draft "2026-03-09-xxx.md" --problem "问题描述" --suggestion "改进建议"
```

### 查看反馈
```bash
aiwriter feedback list --days 7    # 最近 7 天
aiwriter feedback list --all       # 全部
```

### 分析反馈
```bash
aiwriter feedback analyze --days 30  # 分析最近 30 天
```
