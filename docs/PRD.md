# MuseWrite PRD v5

> **项目名称**：MuseWrite - AI驱动的智能写作系统
> **目标**：让创作变得简单而有趣
> **状态**：开发中
> **更新时间**：2026-03-12

---

## 一、产品定位

### 1.1 核心理念

**"灵感驱动，智能写作"**

MuseWrite 是一个 AI 驱动的多平台内容创作工具，帮助创作者：
- 快速将灵感转化为高质量内容
- 一键适配多个发布平台
- 管理个人/团队的内容风格和人设

### 1.2 目标用户

- **个人创作者**：博主、自媒体、KOL
- **内容团队**：品牌运营、MCN机构
- **企业用户**：市场营销、公关部门

### 1.3 核心差异化

| 特点 | 说明 |
|------|------|
| 中文平台优化 | 深度适配小红书、抖音、知乎等中文平台 |
| 四层卡片系统 | 人设卡 + 风格卡 + 平台卡 + 素材卡 |
| 一键多平台适配 | 生成主内容后自动适配各平台格式 |
| 本地优先存储 | 数据在本地，iCloud同步（规划中） |

### 2.2 人设卡（Info Card）

**作用**：定义内容创作者的身份、背景、专业领域

```markdown
# 人设卡 - [名称]

## 基本信息
- 名称：周沫
- 定位：品牌策划人 / 营销顾问
- 标签：#商业洞察 #犀利表达

## 人设描述
专业品牌策划人，10年营销经验，擅长商业洞察和犀利表达，
喜欢用故事讲道理，一针见血不留情面但言之有物。

## 受众画像
- 年龄：25-40岁
- 职业：创业者、产品经理、营销人
- 痛点：商业思维、营销策略、品牌建设

## 语言特点
- 口头禅："说个残酷的真相..."
- 标志性表达：犀利提问 + 案例分析 + 总结升华

## 代表作品（可选）
- 粘贴1-3篇代表作品，AI会学习其写作风格
- 支持导入 Markdown 文件
```

**已实现的人设卡**：
- 周沫（品牌策划）
- 月瑀（治愈系写手）
- 大洋（科技圈老炮）
- 大瑀（生活方式博主）
- 石头哥（个人成长）

### 2.3 风格卡（Style Card）

**作用**：定义内容的调性、语气、结构偏好

```markdown
# 风格卡 - [名称]

## 风格描述
直击要害，一针见血，不留情面但言之有物。

## 语言特征
- 语气：犀利、直接、有力量
- 人称：第二人称为主（"你"）
- 句式：短句为主，善用反问
- 标点：多用省略号、破折号

## 内容结构
1. 开头：抛出痛点/反直觉观点
2. 主体：3-5个论点 + 案例
3. 结尾：总结 + 行动号召

## 禁用词
- 不要用：其实、说实话、不夸张地说

## 风格示例（可选）
- 粘贴符合此风格的示例文本
- 支持导入 Markdown 文件
```

**已实现的风格卡**：
- 犀利（直击要害）
- 温暖（治愈共鸣）
- 专业（严谨客观）
- 幽默（轻松有趣）
- 10个预设风格

```
drafts/              # 草稿存储
tasks/               # 任务队列持久化
logs/                # 日志文件
```

### 4.2 规划方案：本地优先 + iCloud 同步

```
~/Library/Mobile Documents/com~apple~CloudDocs/MuseWrite/
├── personas/        # 人设卡
├── styles/          # 风格卡
├── drafts/          # 草稿
│   ├── 2026-03/
│   │   ├── draft-001.json
│   │   └── draft-002.json
├── templates/       # 自定义模板
└── settings.json    # 设置
```

**优点**：
- 数据完全在用户控制下
- iCloud 自动同步到所有设备
- 类似 Obsidian 的使用体验
- 无需服务器存储

## 五、集成规划

### 5.1 飞书机器人（规划中）

**用途**：团队协作、创作提醒、每日计划

```
飞书群消息 → 飞书机器人 → MuseWrite API → 生成内容 → 推送到群
```

**功能**：
- 每日创作计划提醒
- 内容审核通知
- 发布状态同步
- 快速生成指令

### 5.2 平台发布接口（规划中）

**目标**：一键发布到各平台

| 平台 | 方案 | 难度 |
|------|------|------|
| 小红书 | 官方API/第三方 | 高 |
| 微信公众号 | 官方API | 中 |
| WordPress | REST API | 低 |
| 微博 | 官方API | 中 |
| 知乎 | 官方API | 中 |

### 5.3 N8N 工作流集成

支持通过 N8N 等自动化工具调用 API：

```json
{
  "workflow": {
    "trigger": "定时/手动",
    "steps": [
      { "action": "调用 MuseWrite API 生成内容" },
      { "action": "内容审核" },
      { "action": "发布到平台" },
      { "action": "发送通知" }
    ]
  }
}
```
