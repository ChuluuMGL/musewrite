## 📞 问题排查

### 问题：没有创作主题

**检查步骤**:
1. 检查 Mini/Asuka 是否写了今日总结
2. 检查总结位置是否正确
3. Luna 是否按时执行心跳检查
4. 必要时手动触发总结提炼

### 问题：创作内容风格不对

**检查步骤**:
1. 检查 Creator 是否读取了正确的风格卡片
2. 检查风格卡片内容是否过期
3. 检查 Luna 审核是否到位
4. 必要时更新风格卡片

### 问题：发布失败

**检查步骤**:
1. 检查 AI-Publisher 是否运行正常
2. 检查账号 cookies 是否有效
3. 检查发布配置是否正确
4. 查看发布日志定位问题
