# AI-Writer 卡片规范

> **版本**：v1.0
> **更新时间**：2026-02-28

---

## 一、四层卡片体系

```
素材卡（Source）→ 信息卡（Info）→ 风格卡（Style）→ 平台卡（Platform）
     ↓                  ↓                ↓                ↓
   说什么            谁在说            怎么说            在哪说
```

---

## 二、素材卡（Source Card）

### 模板

```markdown
# 素材卡 - [素材名称]

## 来源
- **类型**：每日笔记 / 灵感记录 / 数据报告
- **时间**：YYYY-MM-DD
- **来源**：Obsidian / 飞书表格

## 内容
- 原始内容

## 标签
- #标签1 #标签2

## 适用账号
- 石头哥 / 周沫 / ...
```

### 示例

```markdown
# 素材卡 - 三层记忆系统改造

## 来源
- **类型**：每日笔记
- **时间**：2026-02-28
- **来源**：Obsidian daily/2026-02-28.md

## 内容
今天完成了三层记忆系统的改造。
三层是 hot/warm/cold。
改造效果是心跳加载从 10KB 降到 2KB。

## 标签
- #AI工具 #OpenClaw #记忆系统

## 适用账号
- 石头哥
```

## 四、风格卡（Style Card）

### 模板

```markdown
# 风格卡 - [风格名]

## 语言风格
- **语气**：轻松 / 专业 / 幽默
- **人称**：第一人称 / 第二人称
- **用词**：口语化 / 书面化
- **特点**：

## 内容结构
- **偏好模板**：清单达成 / 教程 / 对比
- **结构模式**：
  1. 开头
  2. 主体
  3. 结尾

## 标题风格
- **格式**：
- **特点**：

## 禁用词
- 不要用：

## 范文
- 范文1
- 范文2
```

### 示例

```markdown
# 风格卡 - 石头哥风格

## 语言风格
- **语气**：轻松、真诚
- **人称**：第一人称
- **用词**：口语化
- **特点**：短句为主、多用 Emoji、喜欢用"..."

## 内容结构
- **偏好模板**：清单达成
- **结构模式**：
  1. 开头：背景 + ✅ 达成了...
  2. 主体：3-5 个要点
  3. 结尾：Tips + 总结

## 标题风格
- **格式**：✅ 达成了... / 终于搞懂了...
- **特点**：动词开头、积极正面

## 禁用词
- 不要用：首先、其次、最后、综上所述

## 范文
- 达成了 AI 记忆系统的三层改造
- 终于搞懂了 Karpathy 说的"App Store 过时了"
```

## 六、卡片组合规则

### 最小组合

```
素材卡 + 信息卡 + 风格卡 + 平台卡 = 成稿
```

### 多平台

```
素材卡 + 信息卡 + 风格卡 + [平台卡1, 平台卡2, ...] = [成稿1, 成稿2, ...]
```

### 多账号

```
素材卡 + [信息卡1, 信息卡2] + 风格卡 + 平台卡 = [成稿1, 成稿2]
```

## 七、预置风格卡

### 可尝试的风格

| 风格 | 文件 | 适用场景 |
|------|------|----------|
| 幽默风格 | `preset-humorous.md` | 轻松话题、娱乐内容 |
| 故事风格 | `preset-storytelling.md` | 创业故事、成长经历 |
| 专业风格 | `preset-professional.md` | B2B 内容、技术文章 |
| 教程风格 | `preset-tutorial.md` | 工具教程、操作指南 |
| 对话风格 | `preset-conversational.md` | 问答、访谈、观点交流 |

### 使用方式

```bash
# 使用预置风格
/write "素材" --style preset-humorous --platform xiaohongshu
/write "素材" --style preset-storytelling --platform xiaohongshu
/write "素材" --style preset-professional --platform wordpress
```

### 如何选择

| 内容类型 | 推荐风格 |
|---------|---------|
| 工具推荐 | stone-style / preset-humorous |
| 成长故事 | preset-storytelling |
| 行业分析 | preset-professional |
| 教程指南 | preset-tutorial |
| 观点输出 | preset-conversational |

---

_预置风格可用于尝试不同内容风格_
